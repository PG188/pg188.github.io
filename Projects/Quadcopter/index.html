Quadcopter

2014

Project Description

This was a fun project where I aimed to design and construct a quadcopter. All of the CAD work was completed in Autodesk Inventor, with some wind tunnel testing conducted using Autodesk Flow. Unfortunately, the project remained unfinished because I lacked the means to manufacture certain components.

My Role

This was a personal project for which I assumed responsibility. I was tasked with defining the scope, designing, procuring materials, and overseeing the manufacturing of the quadcopter. However, manufacturing posed a significant challenge at the time, ultimately leading to the project's unfinished status.

Reflection

After designing and acquiring all the necessary parts, it was disheartening to discover that I would not be able to machine some of the components due to my limited set of tools. Consequently, the project was abandoned, leaving me with a valuable lesson: always design products with consideration for the available tooling.
