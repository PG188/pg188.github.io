Quadcopter

2014

Project Description

This was just a fun project where i wanted to design and build a quadcopter. All fo the cad work was done in Autodesk Iinventor with some wind tunnel testing on Autodesk Flow. THis project was not completed as I lacked the capability to manufacture some of the parts.

My Role

THis was a persal project. I was responsible for the scope, designing, procuring and manufacturing of the quadcopeter. At the time manufacturing was a huge challenge so the project was not completed.

Reflection

After designing and aquiring all of the parts, it was heartbreaking to find out that i would not be able to machine some of the marts given my limited number of tools. the project has since been abandoned but left me with an important lesson. Always design products based on the tooling immediately available.
